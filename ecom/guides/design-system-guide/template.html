09 — Getting Started

Getting Started Checklist

Follow these steps to turn this template into your working design system.

  1. Fill in your colors
    Hex values for primary, background, accent, and secondary. Add 3+ wrong values to avoid.
  2. Document your fonts
    Families, sizes, weights, usage rules, and @font-face declarations with fallback stacks.
  3. Specify your button component
    Full CSS for default, hover, disabled, and any variants (outline, secondary, etc.).
  4. Set your spacing scale
    Define 4-6 spacing tokens and document what each one is used for.
  5. Write 5 ALWAYS / NEVER rules
    Start with mistakes you've already seen AI make. These are your highest-value entries.
  6. Save as a skill file
    Save to ~/.claude/skills/my-design-system/SKILL.md so Claude Code loads it automatically.
  7. Build a page and run the feedback loop
    Generate a test page, compare to your live site, fix discrepancies, and add new rules. Repeat.
