08 — Do's and Don'ts

Rules at a Glance

Quick reference for brand compliance decisions.

Do
Use RingsideWide as body font (16px)
Use Canela for section titles (sentence case, 38px)
Use #f6f2eb for section backgrounds
Gate hover effects behind .has-mouse
Keep border-radius at 2–3px
Use #000 and #fff as primary pair
Don't
Don't use Geogrotesque for large headlines (use Canela)
Don't use #1A1A1A, #FAF8F5, or #C73A3A
Don't use emojis anywhere
Don't use fully rounded corners (no 9999px)
Don't use viewport-unit typography (no vw)
Don't use brutalist shadows (no 4px 4px 0px)
