Claude Code in one page.

If you've used Claude.ai or Cowork, you've seen Claude answer questions. Claude Code is the same model running in a terminal — except now it can read your files, run commands, and ship code from start to finish.

Loom video coming soon

Three tools, three jobs

  • Claude.ai — personal chat. Great for thinking out loud.
  • Cowork — shared team workspace. Great for collaborating on docs.
  • Claude Code — runs in your terminal. Reads files, writes files, runs commands, calls APIs. It does the work, not just describes it.

Skills are saved workflows

A skill is a Markdown file that tells Claude how to do a specific job. Type /jrb-cro in the terminal and Claude loads that skill's instructions and starts executing. We have ninety-plus skills. Every one is just a file in a folder.

~/.claude/skills/ ├── jrb-cro/SKILL.md ├── jrb-section-builder/SKILL.md ├── jrb-theme-push/SKILL.md └── ...

MCPs connect Claude to other tools

An MCP is a plug that lets Claude talk to an external service. We have MCPs for Slack, Linear, Figma, Air (asset library), Heatmap.com, ListenLabs, Vercel, GitHub, and a dozen more. When you ask Claude to "pull last week's Heatmap data and write it to a doc," it talks to the Heatmap MCP and the file system without any custom code.

Memory makes Claude smarter over time

Claude Code keeps a running memory of facts about you, your team, your projects, and how you like to work. When you correct it ("don't write CSS from scratch"), that becomes a saved memory. The next session it remembers without being told.

The git workflow

Skills, customer intelligence, and pages all live in GitHub repos. Claude can clone, branch, commit, push, and open pull requests. We never edit production code directly — every change is a PR against a develop branch, with QA before merge.

If you want the full intro, the original training is a 19-slide deck at ~/Desktop/claude-code-training/index.html. Cody's already recorded a Loom over it. This page is the cliff notes.

The mental shift

The shift from chat to Claude Code is a shift from asking to delegating. You stop answering yourself and start describing the outcome. The skills do the work. You review and ship.

Try it
Open Terminal. Type claude. Then type /jrb-cro and ask for a test idea for the Miracle Balm PDP. Watch the skill load, read customer intelligence, and produce an ICE-scored recommendation.
