TL;DR

We're testing 4 new layouts of the homepage "Shop By Collection" module against the current live version. No shoot needed. Pull 10 images from Air — 4 lifestyle + 6 multi-hero product, all delivered at 3:4 portrait source (Shopify handles resizing per variant).

01 — What We're Testing

Four new "Shop by Collection" layouts vs. the current live design.

Right now the module is 4 tall lifestyle tiles with an overlay CTA (the Control). The four new variants answer these questions:

Hypothesis 01

Does a "Bloom" tile layout beat the current tall overlay style?

Square image + cream label strip below, vs the overlay CTA we have today.

Hypothesis 02

Does lifestyle or multi-hero product imagery convert better?

Same Bloom layout, different imagery style head-to-head.

Hypothesis 03

Does adding 2 more categories help with discovery?

6 tiles vs. 4 tiles — do customers click more categories when they see more?

Hypothesis 04

If we show 6, should they scroll or all be visible?

Carousel (arrow nav, peek) vs. all-visible grid (Ilia-style).

The winner becomes the new homepage module.

03 — Imagery Style Definitions

Two distinct visual languages to test.

Lifestyle

Model shots, in-use product, people, brand/moment imagery. Soft. Emotional.

Multi-hero Product

Multiple products from the category arranged together (not a single hero SKU). Clean styled still-lifes. Products are the subject. Think the Ilia "Shop By Category" approach.

Examples of multi-hero tiles (reference only, do not copy)

  • Ilia "Complexion" tile = 4–5 complexion products grouped
  • Ilia "Skincare" tile = 3–4 skincare products grouped

06 — Category Content Guidance

What each category tile should feel like.

Face
Complexion products (Miracle Balm, What The Foundation, Miracle Cream, tinted moisturizer).
Body
Body line (the newly launched body collection).
Eyes
Eye products (Just A Sec Shadow, The Best Pencil, mascara).
Lips
Lip products (Cool Gloss, Lip + Cheek Stick, Lippie Stick, Lip Tint).
Skin Care

Bestsellers
Top-selling mix (Miracle Balm + Just a Sec + Cool Gloss + a body or skincare hero). The "greatest hits" feel.
07 — Technical Specs

Source-file requirements.

Color spacesRGB
Minimum source size2000 × 2667 px (3:4 portrait) — Shopify crops to square for V2 automatically
BackgroundClean, brand-appropriate (white, cream, or styled surface). Consistent within a set.
File namingsbc-{set}-{category}.jpg

e.g. sbc-lifestyle-face.jpg, sbc-product-bestsellers.jpg

Shopify will serve resized versions automatically — the team only needs to deliver source files at the minimum size above.

08 — Consistency Rules

Keep the whole module feeling like one shoot.

  • Keep a consistent surface/background within each variant's set (all 4 V1 tiles should feel like the same shoot; all 6 V3/V4 tiles should feel like the same shoot).
  • Product styling should be deliberate — not a random arrangement. Each tile should read instantly as "this category."
  • Fonts/type on tiles: none. The Liquid section renders a cream label strip below every Bloom tile with the category name set in Geogrotesque CondBold (the JRB label font). Don't bake category text into the image.
  • No heavy retouching or overstyling. JRB aesthetic: natural, clean, product-forward.
