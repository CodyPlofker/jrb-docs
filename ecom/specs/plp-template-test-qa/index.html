QA Checklist
All Variants
  • Product cards render correctly at all breakpoints (mobile 375px, tablet 768px, desktop 1280px+)
  • Review counts match PDP counts — not individual product counts (should show group totals e.g. "2.2K Reviews")
  • "X shades" meta line appears for shade products; weight (e.g. "1.2 oz") shows for single-SKU products
  • Apparel shows "X sizes" not "X shades"
  • Save $ flag shows on sale products
  • Sold-out products show gray "Sold Out" button
  • Filter pills work — selecting a category filters the grid correctly
  • Grouped layout (Control / V1 / V2 / V3): group headers visible, correct group order
  • Flat layout (V4): all products in one continuous grid, no group headers
  • No orphan words in any text element at any breakpoint
V1 Only
  • Hover image swaps on desktop card hover
V2 Only
  • Swatch dots appear on cards that have swatch metafields populated
  • Desktop: swatch overlay appears on image hover, positioned at bottom of image
  • Mobile: inline swatch pill visible below title — centered, subtle gray background
  • Clicking or tapping a swatch swaps the card image to that shade
  • Shade description text appears on swatch hover (card meta line)
V3 and V4
  • "Quick Shop" button appears on available non-bundle products
  • Tapping Quick Shop opens modal — bottom sheet on mobile, centered modal on desktop
  • Modal shows: product images/carousel, shade swatches, ATC button, trust messages, description, "See full product details" link at bottom
  • Hovering or tapping a swatch in the modal updates the carousel image
  • ATC button adds to cart and closes modal
  • Close (X) button is visible and functional on mobile (white circle, top-right corner)
  • Bundle products show "Customize" button instead of Quick Shop
  • V4 specifically: flat layout, no group headers, Quick Shop works as expected
Known Items — Not Bugs
  • Filter pill styling — Design team reviewing. Styling may update before launch. Not blocking QA.
  • Missing swatches on some products — e.g. Best Blush shows "5 shades" but no swatch dots. Swatch metafields not yet populated in Shopify admin. Data task, not a code bug.
  • V1 hover image — Currently enabled on V1. Confirm with Cody whether hover should be V1 or V2+ only before launch.
  • Intelligems not wired — Templates are ready; test not yet configured in Intelligems. Links above use direct ?view= routing for preview only.
